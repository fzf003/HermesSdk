# BatchRuns 示例

使用 HermesAgent.Sdk Runs API 的并发批处理演示程序。

## 功能特性

- ✅ **并发启动** — 5 个预定义分析任务同时启动
- ✅ **ANSI 动态仪表盘** — 实时进度条 + 状态图标，原地刷新无滚动
- ✅ **降级模式** — 检测到输出重定向时自动切换为逐行输出
- ✅ **结果汇总** — 成功/失败/取消统计 + 总耗时

## 快速开始

### 1. 配置 API Key

编辑 `appsettings.json`：

```json
{
  "HermesAgent": {
    "ApiBaseUrl": "http://localhost:8642",
    "ApiKey": "your-actual-api-key"
  }
}
```

### 2. 运行程序

```bash
cd samples/BatchRuns
dotnet run
```

## 技术说明

- **ANSI 终端控制**: 使用 `Console.SetCursorPosition` 实现原地刷新
- **线程安全**: `lock` 保护所有控制台写入，避免并行 SSE 流导致的输出错乱
- **进度估算**: `completion`/`error`/`cancelled` → 100%，否则 (事件数 × 20)% 上限 95%
- **重定向检测**: 通过 `Console.IsOutputRedirected` 判断，确保管道和 CI/CD 环境正常运行
